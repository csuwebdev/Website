Javascript, jQuery, and Browser Events

Javascript is used to execute code on the clients machine. User actions are interpreted by the web browser and javascript responds.

  • Browser events allow the computer to monitor the users actions such as the position of your mouse, clicks, keyboard strokes, resizing of windows, page loading, etc.
  • File type of all javascript ends in .js
  • Syntax of a javascript function - example function
  • jQuery is a library of functions written in javascript to help speed up web programming and cross browser compatibility
  • A typical command involves selecting an element, then selecting an event, then selecting an action.

Some examples of things you can do with javascript

  • Change the color of an element on hover
  • Create animations showing, hiding, or changing colors of elements
  • Create interactive menus
  • Create full on animations
  • Create games of any kind

Your Website

If you want a site you need the things below. Begin with the first item, don't worry about any of the others until you have a site in mind!

  • HTML and CSS pages. You can view your website and get it perfect before you launch it online
  • If you want to make your site look even better use Javascript for interactivity
  • If you want to add dynamic content start adding PHP and MySQL
  • Domain Name - Visit Godaddy.com to purchase one for 12$
  • Host - You can find hosting for as low as 1$ a month
